# Wireless Network Watcher Log Viewer
A simple viewer for logs produced by Wireless Network Watcher.

## 📖Introduction
This tool was built to let users browse logs generated by the Wireless Network Watcher Python script. You can view entries ranging from 3 years ago up to the last 15 minutes, or select any custom date and time range.

## 🔤🛠️Languages/Tools
 - C#
 - Microsoft SQL Server (backend)
## 🚀Usage
1. Double-click the executable.
2. Log in to the database server.
3. Select a date/time range.
4. Click Load to view the logs.
